This is the API

getRouteOriginDestination

Get the systems between origin and destination --- This route is cached for up to 86400 seconds

URL

https://esi.evetech.net/v1/route/{origin}/{destination}/

HTTP Method

GET

Response Type

List

Parameters

  • Path: destination Integer

    destination solar system ID

  • Path: origin Integer

    origin solar system ID

  • Query: avoid List(Integer)

    avoid solar system ID(s)

  • Query: connections List(List)

    connected solar system pairs

  • Query: datasource String

    The server name you would like data from

  • Query: flag String

    route security preference

  • Header: If-None-Match String

    ETag from a previous request. A 304 will be returned if this matches the current ETag
