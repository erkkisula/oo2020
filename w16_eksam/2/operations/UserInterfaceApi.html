This is the API

postUiAutopilotWaypoint

Set a solar system as autopilot waypoint ---

URL

https://esi.evetech.net/v2/ui/autopilot/waypoint/

HTTP Method

POST

Response Type

Parameters

  • Query: add_to_beginning Boolean

    Whether this solar system should be added to the beginning of all waypoints

  • Query: clear_other_waypoints Boolean

    Whether clean other waypoints beforing adding this one

  • Query: destination_id Long

    The destination to travel to, can be solar system, station or structure's id

  • Query: datasource String

    The server name you would like data from

  • Query: token String

    Access token to use if unable to set a header
