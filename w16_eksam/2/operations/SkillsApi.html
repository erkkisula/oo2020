This is the API

getCharactersCharacterIdAttributes

Return attributes of a character --- This route is cached for up to 120 seconds

URL

https://esi.evetech.net/v1/characters/{character_id}/attributes/

HTTP Method

GET

Response Type

get_characters_character_id_attributes_ok

Parameters

  • Path: character_id Integer

    An EVE character ID

  • Query: datasource String

    The server name you would like data from

  • Header: If-None-Match String

    ETag from a previous request. A 304 will be returned if this matches the current ETag

  • Query: token String

    Access token to use if unable to set a header
